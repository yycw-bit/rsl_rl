# Extreme Parkour with Legged Robots #

### Installation ###
```bash
pip install -e .
```
- Verify the path
```bash
   python -c "import rsl_rl; print(rsl_rl.__file__)"
```

### About path ###

-**If you replace directly in the default path, do not run pip**

-**If you install in a new path, you need to delete the original path first**
```bash
pip uninstall rsl_rl

```
